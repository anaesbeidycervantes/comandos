Manual de Usuario - Reconocimiento de Órdenes por Voz

Introducción

El sistema de reconocimiento de órdenes por voz permite controlar diversas acciones en el navegador utilizando comandos de voz simples. Este manual de usuario proporciona una guía detallada sobre cómo utilizar el sistema y los comandos disponibles.

Cómo Usar

Para utilizar el sistema de reconocimiento de órdenes por voz, sigue estos pasos:

  1. Abre el navegador web en tu dispositivo.
  2. Dirígete a la página web del sistema.
  3. Asegúrate de que el micrófono de tu dispositivo esté activado.
  4. Una vez en la página web, espera a que aparezca la indicación para empezar a hablar.
  5. Di uno de los comandos disponibles en voz alta y clara.
  6. El sistema reconocerá tu comando y realizará la acción correspondiente.

Comandos Disponibles

Los siguientes son algunos de los comandos disponibles que puedes utilizar con el sistema de reconocimiento de órdenes por voz:

  • abrir Google: Abre Google en una nueva pestaña del navegador.
  • abrir Netflix: Abre Netflix en una nueva pestaña del navegador.
  • instrucciones: Abre una nueva pestaña con el manual de usuario (este documento).
  • buscar Morat: Realiza una búsqueda en el navegador con el término "Morat".
  • buscar Pachuca: Realiza una búsqueda en el navegador con el término "Pachuca".
